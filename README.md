# Halcón WebApp

## Descripción
Aplicación web para la gestión de pedidos de una distribuidora de materiales de construcción.

## Objetivo
Automatizar el proceso de pedidos y permitir el seguimiento de su estado por parte de los clientes.

## Tecnologías
- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Funcionalidades
- Registro de pedidos
- Consulta de pedidos
- Gestión de usuarios
- Gestión de roles
- Evidencias fotográficas
- Dashboard administrativo
